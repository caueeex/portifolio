Chatbot para Atendimento SENAI

Sistema de atendimento automatizado via WhatsApp para otimizar a comunicação com clientes de forma prática e eficiente.

Tecnologias:

Node.js Express Socket.IO PHP MySQL

Funcionalidades:

  • Bot no WhatsApp: Respostas automáticas sobre serviços, suporte e documentos
  • Atendimento humanizado: Transição para equipe humana quando necessário
  • Painel administrativo em PHP + MySQL para gerenciar atendimentos
  • Mensagens rápidas pré-definidas

Impacto:

  • Atendimento mais rápido e eficiente
  • Melhor experiência para usuários e equipe
  • Automação sem perder o toque humano
